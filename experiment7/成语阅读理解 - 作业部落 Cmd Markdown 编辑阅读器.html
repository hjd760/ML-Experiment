成语阅读理解

机器学习 实验

实验目的

  1. 探索自然语言处理
  2. 理解BERT的原理
  3. 熟练对BERT的运用
  4. 在简单小规模数据集上实现阅读理解，培养工程能力

数据集描述

  1. 采用ChID数据集(提取码为 3en8 )
  2. 每条数据由若干段文本和一组固定长度的候选项构成，每段文本被挖去了若干个空格（每个空格都有唯一的编号），需要从候选项中选出每个空的答案。注意同一条数据的文本的填空答案在词义或语境上可能是相近的。保证每一条数据中，各个空的答案互不相同。

实验环境

python3，至少包含下列python包：sklearn，numpy，jupyter，matplotlib。
建议直接安装anaconda3，其已经内置了以上python包。

自行安排

不要求

实验步骤

本次实验代码在PyCharm上完成。

熟练掌握BERT的运用

  1. 下载数据集和代码（https://github.com/chujiezheng/ChID-Dataset/tree/master/Competition）（BERT-based Baseline）。
  2. 根据BERT-based Baseline github 的指示下载BERT模型和配置环境。
  3. 修改代码输出预测的成语。

整理实验结果并完成实验报告。
